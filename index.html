BETA DEMO // 固定主砲與夥伴整備版

霓虹拾荒者 Neon Salvage

v7.9 固定主砲、縮小子彈比例；護甲 / 增效 / 磁吸道具回歸，暫停與結算可查看主機與夥伴技能樹。

v7.9 固定主砲與夥伴整備版
  • 主武器固定為主砲，不再因撿到掉落物而變成其他武器。
  • 玩家與敵方子彈尾焰、碰撞半徑與視覺比例縮小，配合 v7.8 小主機尺寸。
  • 擊殺會掉落維修、貨幣、護甲、增效、磁吸等道具，強化本局戰鬥節奏。
  • 暫停 / 結算 / 首頁可看主機永久技能樹；暫停可看夥伴機技能樹並消耗重複取得的技能點。
  • 無人夥伴機改成環繞主機巡航攻擊 / 支援，不再像黏在主機身上。
10 波撤離短局 roguelite 生存 夥伴機任務停在區塊旁開啟支援機 主機技能樹用貨幣永久升級 手機可玩單手觸控與自動瞄準 
Beta Demo 速讀

目標：
- 撐過 10 波敵群與 Boss，途中收集碎晶、道具與夥伴機。
- 第 5 波是 Boss 檢查，第 10 波擊破星環主宰即可撤離成功。
- v7.9 重點：武器固定主砲、子彈比例縮小；掉落物改為護甲 / 增效 / 磁吸 / 維修 / 貨幣。

操作：
- WASD / 方向鍵：移動。
- 滑鼠：控制機身朝向；自動瞄準會輔助鎖最近敵人。
- E：切換自動瞄準：輔助 → 完全 → 關閉。
- 空白：相位衝刺，短時間無敵。
- P 或畫面上的「整備」：暫停並開啟主機 / 夥伴技能樹。
- 設定：操作模式、音量、震動、難度與測試音效。

手機：
- 切到手機模式後，戰鬥中按下哪裡，浮動方向鍵就會在該處出現。
- 拖曳方向就是推進方向；放開後方向鍵會隱藏並清除移動輸入。
- 手機模式建議使用自動瞄準；若手機沒聲音，先到設定點「測試音效」。

戰鬥重點：
- 主武器固定為主砲，不會再因掉落物變成霰彈、穿甲或環刃。
- 擊殺敵人會掉落維修包、貨幣箱、護甲板、增效核心與磁吸增幅。
- 護甲板短時間減傷；增效核心短時間提高主砲傷害與射速；磁吸增幅提高拾取範圍。
- 3.6 秒內 5 連殺會觸發「擊破爆發」，用金色衝擊波清附近敵人。
- 任務區塊會出現在地圖上，靠近並停住可開啟無人夥伴機。
- 無人夥伴機會環繞主機巡航攻擊 / 支援，不會黏在主機身上。
- 重複取得同型夥伴機會獲得技能點，可在暫停 / 結算整備中升級該機專屬技能樹。
- 暫停、首頁或結算可以花碎晶升級主機永久技能樹，例如主砲線圈、反應爐、護盾、裝甲、推進、拾荒磁場與夥伴連線。
- 失敗後可花碎晶接關；若選擇重新出擊，夥伴機與本局技能點會歸零，主機永久技能保留。
