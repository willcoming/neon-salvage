BETA DEMO // 10 波撤離制 roguelite

霓虹拾荒者 Neon Salvage

駕駛小型飛船穿越破碎星環：完成目標、撐過事件、組出技能流派，擊破第 10 波星環核心後撤離。這是可公開試玩的 Beta Demo，適合用桌機或手機快速體驗一局。

v6.0 效能診斷與模組化基礎版
  • 新增 F3/設定可切換的效能面板，顯示 FPS、frame/update/draw timing 與場上 entity 數量。
  • save migration、balance/pacing helper 與 diagnostics 已拆成可測模組，後續調參更安全。
  • 保留 v5.9 擦彈機動、桌機/手機操作、音效設定、永久升級與 run report 相容性。
10 波撤離短局 roguelite 生存 Build 流派三選一技能組核心 事件 / 戰術 / Boss讀提示破解壓力 手機可玩單手觸控與自動瞄準 
Beta Demo 速讀

目標：
- 完成地圖目標、撐過事件、組出技能流派。
- 第 5 波是 Boss 檢查，第 10 波擊破「星環核心主宰」即可撤離成功。
- 結算會給 C/B/A/S 評級，並列出 Build、核心超載、擦彈機動、契約 / 詛咒、局內抉擇、路線後果、Boss 前預備、Boss 改造、壓力、目標路線、戰術破解與下一局建議。

操作：
- WASD / 方向鍵：移動。
- 滑鼠：控制機身朝向與手動瞄準。
- E：切換自動瞄準：輔助 → 完全 → 關閉。
- 空白：相位衝刺，短時間無敵。
- P：暫停並開啟艦載升級。
- F3：切換效能面板（FPS、frame/update/draw timing 與場上 entity 數量）。
- 設定：操作模式、音量、震動、難度、效能面板與測試音效。

手機：
- 切到手機模式後，按住螢幕會出現浮動方向鍵。
- 手機模式預設使用自動瞄準，沒有衝刺鍵。
- 手機版任務 HUD 是單條橫列；平常會淡出，事件、目標或任務變化時會短暫亮起。
- 若手機沒聲音，先到設定點「測試音效」喚醒音訊。

畫面閱讀：
- 左上 P1：目前 run 節奏、本局異變任務、開局契約 / 詛咒、局內路線、路線後果與 Boss 前預備。
- 左上 P2：當前事件、核心超載、擦彈機動、敵群戰術、Boss 讀題、反制提示與破解加成。
- 左上 P3：目前目標與副條件。
- 地圖標籤：維修、新星、超頻、危險區、補給、資源。

異變 / 契約玩法：
- 開局契約 / 詛咒會改變實際數值：過載合約偏射速、玻璃核心偏高風險火力、拾荒債券偏資源、護航協議偏生存。
- 第 2 / 第 4 波會出現 2 選 1 局內抉擇：安全補給、高風險懸賞、弱點掃描、晶礦開採等路線會改變後續事件池與 Boss 壓力。
- 選路線後會出現小型後果任務：補給護送、懸賞菁英、掃描中繼、晶礦鑽探；完成給 Tempo Boost 與資源，錯過會寫入結算。
- 完成後果會變成 Boss 前預備：防禦補給艙、懸賞弱點、讀題校準、晶礦賭注會分別改變下一個 Boss 的彈幕、生命、破招窗口或掉落與伏擊。
- 碎晶潮汐：短時間內連撿碎晶，達標後給超頻核心。
- 懸賞獵場：擊破菁英累積懸賞，達標後給新星炸彈。
- 裂隙干擾：地圖會出現裂隙封印點，靠近充能可清掉危險區。
- 補給航道：地圖會出現補給艙，靠近護送充能並避免敵人破壞。

地圖與目標：
- 藍色目標點會預告「目標 → 事件 → 完成後加成」。
- 路線抉擇節點會以雙菱形與充能 ring 顯示；靠近其中一個充滿後，另一個會消失。
- 路線後果會以補給艙、掃描中繼、晶礦鑽探圈或「賞」標記菁英呈現，藍色箭頭會優先指向它。
- 副條件完成會標 ★，給額外獎勵。
- 不同目標有不同連鎖效果：掃描給弱點標記、守點給補給、採集噴碎晶、裂隙清危險區、獵殺給懸賞。
- 星域可選隨機、電磁殘骸帶、晶礦雲帶、裂隙邊界；每個星域會帶不同 Boss modifier。

戰鬥重點：
- 先撿碎晶、拿技能，盡快讓一個 Build 成形。
- Build 核心成形後，船身光環、子彈尾焰與命中火花會變成該流派顏色；連續擊破會累積核心連殺並觸發短暫「核心超載」。
- 技能卡會顯示選後分數與推薦理由；優先看「核心候選」與目前缺口。核心成形後，連殺超載會依流派給射速、火力、磁吸或續航加成。
- 敵彈剛好擦身而過會累積擦彈；3 次擦彈會觸發「擦彈機動」，短時間提升移速、射速並降低受傷。
- 事件完成後會有短暫 Tempo Boost；看到 HUD 顯示加成時可以主動進攻或撿資源。
- 追獵機會包抄，別只往後退；橫向走位能拉開角度。
- 閃擊機會先預備再突進，看到橘色突進線可側移反打。
- 狙擊球會保持距離，看到粉色瞄準線要橫向移動。
- 爆裂雷接近後會倒數，橘色爆炸圈是安全距離提示。
- 受傷時紅色方向提示會指出傷害來源；碎晶被吸附時會有黃色拖尾。
- 護盾衛星、治療者、加速單位、狙擊球、折射吸能蟲是戰術關鍵；擊破後會觸發短暫戰術破解加成。
- Boss 出招前會出現讀題提示與蓄力 ring；進入二階段時會有 PHASE SHIFT 轉場與 phase ring。
- Boss 會套用星域改造與已完成的 Boss 前預備：例如 EMP 裝甲較硬但彈幕慢、讀題校準讓破招窗口更長、晶礦賭注提高掉落但帶來高速敵伏擊。
- 流星落點、召喚裂縫、脈衝安全縫會直接畫在戰場上；穿過脈衝安全縫會觸發短暫「穿縫反擊」。
- 擊破第 10 波星環核心會觸發 SECTOR CLEAR 爆光與碎晶雨，結算會列出 Boss 擊破亮點。
- 第 9 波有終局整備補給；拿完再打第 10 波核心。
- 失敗也會保存資源，可用艦載升級強化下一局。
