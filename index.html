  • 首頁改成公開展示用 landing：清楚標示 Beta Demo、10 波撤離、桌機 / 手機皆可玩。

  • 保留完整 Beta 內容：星域路線、地圖事件、敵群戰術、Boss 二階段、build 流派、結算報告與手機 UI。
10 波撤離短局 roguelite 生存 Build 流派三選一技能組核心 事件 / 戰術目標會接地圖壓力 手機可玩單手觸控與自動瞄準 
Beta Demo 速讀

目標：
- 完成地圖目標、撐過事件、組出技能流派。
- 第 5 波是 Boss 檢查，第 10 波擊破「星環核心主宰」即可撤離成功。
- 結算會給 C/B/A/S 評級，並列出 Build、壓力、目標路線、Boss 機制與下一局建議。

操作：
- WASD / 方向鍵：移動。
- 滑鼠：控制機身朝向與手動瞄準。
- E：切換自動瞄準：輔助 → 完全 → 關閉。
- 空白：相位衝刺，短時間無敵。
- P：暫停並開啟艦載升級。
- 設定：操作模式、音量、震動、難度與測試音效。

手機：
- 切到手機模式後，按住螢幕會出現浮動方向鍵。
- 手機模式預設使用自動瞄準，沒有衝刺鍵。
- 若手機沒聲音，先到設定點「測試音效」喚醒音訊。

畫面閱讀：
- 左上 P1：目前 run 節奏，例如暖機 / Build 成形 / 高壓選擇 / 終局考驗。
- 左上 P2：當前事件與敵群戰術。
- 左上 P3：目前目標與副條件。
- 地圖標籤：維修、新星、超頻、危險區、補給、資源。

地圖與目標：
- 藍色目標點會預告「目標 → 事件」。
- 副條件完成會標 ★，給額外獎勵。
- 星域可選隨機、電磁殘骸帶、晶礦雲帶、裂隙邊界。

戰鬥重點：
- 先撿碎晶、拿技能，盡快讓一個 Build 成形。
- 護盾衛星、治療者、加速單位要優先拆。
- 第 9 波有終局整備補給；拿完再打第 10 波核心。
- 失敗也會保存資源，可用艦載升級強化下一局。
